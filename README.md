PPAsyncDrawingKit
------------------------
这是一个轻量的异步绘制框架，实现一系列 `ImageView`, `Label`, `Button` 等基础控件，支持 `Objective-C`, `Swift`。

#Features
* 使用多线程技术进行绘制，保证复杂界面的流畅
* 不阻塞线程
* 轻量，接近 `UIKit` API，使用简单
* 排版，渲染 `AttributedString`
* 异构 `TextView`，避免创建大量 `Label`
* 高性能圆角图片
* 保持 FPS 在低端设备上稳定在 60，减小抖动


#Requirements
* iOS 7.0 or later
* Xcode 7.3 or later
